# Heart Disease Analysis Dashboard

This project analyzes heart disease data using SQL queries and visualizes insights using Tableau dashboards and stories.  
The dashboard is integrated with a Flask web application to display the visualization in a web interface.

## Project Features
- Data analysis using SQL queries.
- Interactive Tableau dashboard
- Story-based visualization explaining insights
- Flask web integration for displaying the dashboard

## Technologies Used
- Python (Flask)
- Tableau Public
- SQL
- HTML

## How to Run the Project

1. Clone the repository

git clone https://github.com/rishikanigam2006315/Heart_disease_analysis.git

2. Navigate to the project folder

cd Heart_disease_analysis

3. Run the Flask application

python app.py

4. Open the browser and go to

http://127.0.0.1:5000
